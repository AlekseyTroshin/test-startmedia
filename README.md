# test-startmedia
Задания выполнены в двух вариантах на чистом php и php-js

### PHP (основное)
в фале php

* запускаем composer install
* php -S localhost:8000 'public/index.php'

### PHP-JS
в файле php-js

* заходим в servers
* запускаем composer install
* php -S localhost:8000 'public/index.php'
* в папке client открываем index.html
